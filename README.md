# Sales Data Analysis

## Overview
This project analyzes sales data using Python.

## Skills
- Python
- pandas
- matplotlib
- Data Analysis

## Description
This project reads sales data from a CSV file, calculates total sales by month and by product, and visualizes the results with charts.

## Results
- Product A recorded the highest total sales.
- Monthly sales showed an overall upward trend.

## How to Run
1. Install Python
2. Install required libraries:
   pip install pandas matplotlib
3. Run:
   python main.py
